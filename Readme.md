### Airplane Tracker System
# Система для отслеживания самолётов с интеграцией API и хранением данных.

**Описание**
Проект представляет собой систему для отслеживания самолётов, которая:

* получает данные о самолётах через `API (OpenSky Network);`

* ищет географические данные через `Nominatim API;`

* хранит информацию о самолётах в `JSON‑файлах;`

* предоставляет гибкий интерфейс для фильтрации и управления данными.

# Основные компоненты
**Класс `Airplane`**
Представляет информацию об одном самолёте. Поддерживает:

* валидацию всех входных данных;

* сравнение самолётов (по высоте, затем по скорости);

* вспомогательные методы:

- `get_speed_kmh()` — конвертирует скорость в км/ч;

- `is_in_flight()` — проверяет, находится ли самолёт в полёте (высота > 100 м);

- `to_dict()` — сериализует объект в словарь.

**API‑классы**
* `NominatimAPI` — работа с `OpenStreetMap Nominatim` для поиска географических данных.

* `OpenSkyAPI` — получение данных о самолётах из OpenSky Network.

Оба класса наследуются от абстрактного BaseAPI.

**Хранилище данных**
`JSONFileConnector` — реализация хранилища на основе JSON‑файлов, наследуется от BaseStorageConnector. Поддерживает:

* добавление, удаление и обновление самолётов;

* фильтрацию по различным критериям;

* сохранение и загрузку данных из файла.
